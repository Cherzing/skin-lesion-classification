# 消融实验设计

## 实验目的

验证“毛发抑制引导的双流度量分类网络”中各关键组成部分的独立贡献，明确模型性能提升是否来自毛发抑制引导、双流结构、CBAM 注意力、度量约束分类头与差分引导机制。

## 完整模型

完整主模型包含以下机制：

1. 毛发抑制引导输入：第二分支使用去毛发图像。
2. 双流结构：原图分支与去毛发分支并行特征提取。
3. 差分引导机制：利用差分图生成动态融合权重与空间引导。
4. CBAM 注意力增强。
5. 度量约束分类头。

## 消融变体定义

### 1. 无毛发抑制引导

- 设置：保留双流结构，但第二分支不再使用去毛发图，而改为重复输入原图。
- 目的：验证去毛发信息本身是否对模型有独立贡献。

### 2. 无双流结构

- 设置：仅保留原图单分支，移除双流融合。
- 目的：验证双流设计是否优于单流结构。

### 3. 无 CBAM

- 设置：保留双流与差分引导，但关闭注意力增强模块。
- 目的：验证注意力机制对关键病灶特征提取的帮助。

### 4. 无度量约束分类头

- 设置：保留双流与引导结构，但将度量约束分类头替换为普通线性分类输出。
- 目的：验证角度间隔约束对“异病同像”问题的贡献。

### 5. 无差分引导

- 设置：保留双流结构，但去除差分图引导，改为固定双分支平均融合。
- 目的：验证差分先验对动态融合策略的作用。

## 统一实验设置

- 数据划分：固定使用 `train / val / test`
- 输入尺寸：`160 x 160`
- epoch 数：`5`
- batch size：`8`
- 学习率：`1e-4`
- 初始化方式：统一采用旧版模型骨干热启动

## 评价指标

- Accuracy
- Macro Precision
- Macro Recall
- Macro F1

## 预期验证问题

1. 去除任何一个关键模块后，模型整体性能是否下降。
2. 哪个模块对整体性能影响最大。
3. 毛发抑制引导与差分引导在模型中是否发挥不同层次的作用。
